# 域名购买 + 部署 + 表单对接指南

---

## 一、域名购买（推荐 Namecheap）

### 最简单的方案
1. 打开 https://www.namecheap.com
2. 搜索你想要的域名，比如 `jiufupackaging.com`
3. 建议选 `.com`，价格约 **$10-14/年**
4. 加入购物车，注册账户，PayPal/信用卡付款
5. 域名就到手了

### 其他域名平台
| 平台 | 特点 |
|------|------|
| Namecheap | 推荐，便宜好用，支持支付宝 |
| GoDaddy | 全球最大，稍贵 |
| Cloudflare Registrar | 最便宜（成本价），需绑信用卡 |
| 阿里云万网 | 国内方便，`.com` 约55元/年 |

## 二、网站部署（推荐 Netlify，免费）

### 免费方案：Netlify（5分钟搞定）
1. 打开 https://www.netlify.com
2. 用 GitHub 账号注册登录
3. 把 `jiufu-website` 文件夹拖拽到 Netlify 页面
4. 自动部署，获得一个免费域名如 `xxx.netlify.app`
5. 在设置里绑定你买的域名

### 国内方案：阿里云 OSS + CDN
1. 阿里云开通 OSS（对象存储），创建 Bucket
2. 设置为静态网站模式
3. 把所有 HTML/CSS/JS 文件上传到 Bucket
4. 绑定域名，配置 CDN 加速
5. 成本：几块钱/月

## 三、表单后端对接

### 方案一：PHP 发邮件（已提供文件）

文件 `send-mail.php` 已经生成在网站文件夹里。

**使用步骤：**
1. 确保你的主机支持 PHP（cPanel/虚拟主机都有）
2. 把 `send-mail.php` 一起上传到服务器
3. 修改 contact.html 中的表单 action：
```html
<form action="send-mail.php" method="POST">
```
4. 确保 JS 里的 `e.preventDefault()` 注释掉，让表单正常提交

### 方案二：Formspree（零配置，推荐新手）

1. 打开 https://formspree.io
2. 注册免费账户（每月50封邮件）
3. 创建新表单，获得一个 URL 如 `https://formspree.io/f/xxxxx`
4. 把 contact.html 表单 action 改为这个 URL：
```html
<form action="https://formspree.io/f/你的ID" method="POST">
```

### 方案三：对接网易企业邮箱 SMTP

如果要用 wzjiufu@163.com 发信，可以在 send-mail.php 里改用 SMTP：
```php
// 用 PHPMailer 库，或直接用 163 SMTP
// SMTP: smtp.163.com, 端口 465 (SSL)
// 用户名: wzjiufu@163.com
// 密码: 邮箱的授权码（不是登录密码！）
```

授权码获取：登录 mail.163.com → 设置 → POP3/SMTP/IMAP → 开启 → 获取授权码
